Формальные сведения о проекте
Название проекта
Разработка сервиса автоматизированного обучения RAG-моделей (AI-as-a-Service)
Тип проекта
Проектная деятельность
Предметная область
Искусственный интеллект, интеллектуальный поиск информации, генеративные языковые модели
Цель проекта
Создание корпоративной платформы для интеллектуального поиска знаний и автоматизированного обучения RAG-моделей.
Основные задачи
  • Проанализировать существующие решения в области RAG.
  • Разработать архитектуру платформы.
  • Настроить векторное хранилище данных.
  • Интегрировать языковые модели.
  • Провести тестирование решения.
Используемые технологии
OpenWebUI, Ollama, Gemma, ChromaDB, RabbitMQ, Docker
Команда проекта
Ярослав Дорофеев, Карина Осадчая, Виталий Харитонов, Каролина Шишуева, Ярослав Юматов
Ожидаемый результат
Рабочий прототип платформы AI-as-a-Service для поиска знаний на основе Retrieval-Augmented Generation.
